Youtube 악보 캡처

유튜브 탭/창을 화면 공유한 뒤, 악보 영역만 지정하세요.

대기 저장: 0장
영역: 미지정
선택: 0장

PWA: 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.

  1. 유튜브에서 악보 영상을 재생합니다.
  2. 화면 공유 → 유튜브 탭/창 선택 (모바일은 지원이 제한될 수 있음)
  3. 미리보기에서 악보만 드래그
  4. 시작 후 영상이 끝나면 PDF 생성
  5. 캡처 기능은 PC Chrome/Edge 권장 (아이폰 Safari 미지원)
  6. 갤럭시: Chrome 메뉴 → 홈 화면에 추가 / 앱 설치
  7. 아이패드: Safari 공유 → 홈 화면에 추가 (설치만 가능, 캡처는 PC 권장)
화면 공유를 시작하면 미리보기가 표시됩니다.
썸네일을 눌러 선택 (여러 장 가능)
